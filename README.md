# Open Source Platform Analysis

Analysis of open source platforms including Galaxy and GitHub data, exploring contributions, downloads, categories, and collaboration patterns.

## Project Structure

```
.
├── analysis/          # Generated visualizations and analysis outputs
├── data/             # Raw and processed data files
│   ├── galaxy_data/
│   ├── github_data/
│   └── github_company_data/
└── scripts/          # Jupyter notebooks for analysis
    ├── galaxy_platform_anaylsis.ipynb
    ├── github_analysis_1.ipynb
    ├── github_analysis_2.ipynb
    └── github_companies_anaylsis.ipynb
```

## Analysis Outputs

The project generates various visualizations including:
- Category distribution analysis
- Download metrics and patterns
- Company type contributions
- Global collaboration maps
- Owner and category heatmaps
- Interactive sunburst and treemap charts

## Getting Started

1. Clone the repository
2. Install required dependencies (Python, Jupyter, pandas, matplotlib, plotly, etc.)
3. Open the Jupyter notebooks in the `scripts/` directory
4. Run the analysis notebooks to generate visualizations

## Requirements

- Python 3.x
- Jupyter Notebook
- pandas
- matplotlib
- plotly
- seaborn
- numpy
